# 企业AI工作流平台 · AI Command Center

一个面向传统企业的 **AI 中控指挥舱 Demo**，专注于帮助企业安全、高效地落地AI能力。

![项目预览](https://v0-ai-drab-rho.vercel.app/og-image.jpg)  
**在线预览**：https://v0-ai-drab-rho.vercel.app/

## 项目背景

传统企业在推进AI落地时，普遍面临以下痛点：
- 数据安全与合规风险高（敏感客户资料不敢上云）
- 多Agent协作混乱，缺乏统一调度
- 知识碎片化，经验无法有效沉淀
- 难以量化AI带来的真实ROI

本项目正是针对这些问题设计的**企业级AI工作流中控平台**。

## 核心功能

### 1. 经营总览仪表盘
- 实时展示AI带来的业务价值（节省工时、转化率提升、知识命中率等）
- 多维度数据趋势可视化

### 2. AI能力全景监控
- **AI客服**：智能意图识别 + 知识库回答
- **员工Copilot**：内部知识检索与工作辅助
- **知识中台**：RAG增强 + 知识回写机制
- **多Agent协同**：工蜂-兵蜂-蜂后分层架构
- **工作流自动化**：可视化流程编排

### 3. 技术架构亮点
- **100%本地私有化部署**（支持Docker）
- **高精度RAG知识库**：有效降低幻觉
- **多模型动态路由**：Gemma 4 + Qwen 3.6 + GLM-5 协同
- **夜间知识蒸馏**：对话自动沉淀为企业资产
- **人机协同闭环**：人工兜底 + 知识回写

## 技术栈

- **Frontend**：Next.js 15 + Tailwind CSS + shadcn/ui
- **AI相关**：多Agent架构、RAG、Prompt Engineering
- **部署**：支持 Docker 私有化部署 / Vercel
- **设计工具**：v0.dev

## 设计理念

**「群星系统」** —— 采用蜂群式分层架构（工蜂执行 + 兵蜂复盘 + 蜂后调度），结合蚂蚁信息素机制实现跨部门经验自动涌现，在**尊重企业现有管理层级**的同时，显著提升协同效率和知识复用率。
